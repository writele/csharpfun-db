One-to-Many

Project
    public class Project
    {
        public Project()
        {
            this.Tickets = new HashSet();
        }

        public virtual ICollection Tickets { get; set; }
    }
Ticket
    public class Ticket
    {

      public int ProjectId { get; set; }

      public virtual Project Project { get; set; }

    }
The Database Relationship

In our database, each entry in our Tickets table has a Foreign Key that points to the Primary Key in our Projects table.

The Object Relationship

We can call a list of all the tickets that belong to a specific project:

          var project = db.Projects.Find(7);
          var tickets = project.Tickets.ToList();

We can also call an object that will represent the project that a ticket belongs to:

          var ticket = db.Tickets.Find(12);
          var project = ticket.Project;
